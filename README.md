# Jade Savoia — Portfolio

A personal design portfolio site for Jade Savoia — a digital creative focused on accessible, engaging experiences.

## Structure

- `index.html` — Homepage (hero, audience switcher, featured work, about, reviews, contact)
- `project.html` — Case study: Activate Games (discovery & strategy)
- `project-2.html` — Case study: Etsy iOS (information architecture)
- `project-3.html` — Case study: Church consolidation & design system
- `project-4.html` — Case study (template)
- `styleguide.html` — Design system / style reference
- `tokens.css` — Design tokens (color, type, spacing, motion)

## Tech

Static HTML/CSS with light vanilla JavaScript for scroll reveals, the audience copy switcher, carousels, and animated charts. No build step — open `index.html` in a browser.

## Local preview

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
